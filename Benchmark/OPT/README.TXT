Questa cartella contiene due codici python per il Reverse Design.
Assicurarsi che il prodilo NACA0012.dat sia nella cartella di lavoro per entrambi i codici.
Per l'utilizzo del modello surrogato, assicurarsi che anche il best_model.keras sia presente dentro la cartella di lavoro.

Per girare deve essere presente nella cartella di lavoro un file testo denominato params.txt che cotiente 20 valori da utilizzare come parametri WG2AER + 1 AoA
Ad ogni run viene restituito in output un file contenente RMSE rispetto al target ricercato, coordinate asse X ed Y ed il valori del CP per la configurazione testata.
